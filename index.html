SetupConfiguração

Running in under an hour.Rodando em menos de uma hora.

Five steps. No servers to manage. GitHub Actions runs everything on schedule, for free. Cinco etapas. Sem servidores. GitHub Actions roda tudo no schedule, de graça.

Or follow our interactive setup guide for a step-by-step walkthrough. Ou siga nosso guia interativo de setup para um passo a passo completo.

01
Clone the repo & run vera setup
Clone o repo & rode vera setup
▾

Fork veralifeos/vera-open. Run vera setup: the wizard walks you through every credential: Notion, Telegram, Claude API, JSearch.

Fork veralifeos/vera-open. Rode vera setup: o wizard te guia em cada credencial: Notion, Telegram, Claude API, JSearch.

Run vera doctor to validate your environment before first run.

Rode vera doctor para validar seu ambiente antes da primeira execução.

02
Set up your Notion template
Configure seu template Notion
▾

Duplicate the official Vera Notion template. Tasks DB, Job Pipeline, Weekly Check-in, Linha do Céu, all pre-structured.

Duplique o template oficial Vera no Notion. Banco de Tarefas, Pipeline de Vagas, Check Semanal, Linha do Céu, tudo pré-estruturado.

03
Configure your Research Packs
Configure seus Research Packs
▾

Edit research.yml to enable packs and set schedules. Each pack has a notify_min score threshold; results below it are silently filtered.

Edite research.yml para ativar packs e definir schedules. Cada pack tem um threshold notify_min; resultados abaixo são filtrados em silêncio.

Use vera packs list to see available packs and their status. Use vera validate to check your config before deploying.

Use vera packs list para ver packs disponíveis e seu status. Use vera validate para checar seu config antes de fazer deploy.

04
Add secrets to GitHub Actions
Adicione secrets ao GitHub Actions
▾

Copy your .env values as GitHub repository secrets. The workflow picks them up automatically, no code changes needed.

Copie os valores do seu .env como secrets do repositório GitHub. O workflow os captura automaticamente.

05
First briefing in your Telegram
Primeiro briefing no seu Telegram
▾

Trigger manually: vera run --briefing. If green, Vera runs automatically at your configured schedule. Done.

Dispare manualmente: vera run --briefing. Se tudo verde, Vera roda automaticamente no seu schedule. Pronto.
