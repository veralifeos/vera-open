SetupConfiguração

Set up in under an hour.Configurado em menos de uma hora.

Vera was designed to be installed by real people, not just developers. The process takes under an hour and you won't be doing it alone: alongside detailed documentation available after sign-up, there's a GPT-powered Setup Assistant that guides you step by step, from clone to your first briefing delivered on Telegram. Vera foi pensada para ser instalada por pessoas reais, não só por devs. O processo leva menos de uma hora e você não precisa fazer isso no escuro: além de uma documentação detalhada acessível após o cadastro, há um Assistente de Instalação disponível via GPT que te guia passo a passo, do clone ao primeiro briefing entregue no Telegram.

01
Clone the repository
Clone o repositório
▾

One terminal command and you have the project locally. No advanced experience needed — just an open terminal.

Um comando no terminal e você tem o projeto localmente. Não precisa de experiência avançada, só de um terminal aberto.

02
Set up your credentials
Configure suas credenciais
▾

Create a .env file with your tokens: Notion, Telegram and Claude API. The GPT Assistant shows you exactly where to find each one.

Crie um arquivo .env com seus tokens: Notion, Telegram e Claude API. O Assistente GPT te mostra exatamente onde encontrar cada um.

03
Run the interactive setup
Rode o setup interativo
▾

The vera setup command opens a wizard that automatically creates the databases in Notion and validates all connections before proceeding.

O comando vera setup abre um assistente que cria automaticamente os bancos de dados no Notion e valida todas as conexões antes de continuar.

04
Test with a dry run
Teste com um dry-run
▾

Before activating real delivery, run a test briefing. You see exactly what would be sent to Telegram without committing any state.

Antes de ativar o envio real, rode um briefing de teste. Você vê exatamente o que seria entregue no Telegram sem comprometer nenhum estado.
